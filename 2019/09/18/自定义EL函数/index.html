自定义EL函数

EL总结

  • 不能出现在 Java 代码块、表达式块等 JSP 的动态代码部分。
  • 只能从 pageConext、request、session、application 四大域属性空间中获取数据。
  • 不会抛出空指针异常。若访问一个 null 对象的属性，则什么也不显示。
  • 不会抛出数组访问越界异常。若访问一个数组中的不存在的元素，则什么也不显示。
