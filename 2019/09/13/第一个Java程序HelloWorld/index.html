Java开发软件介绍

工欲善其事必先利其器，要编写Java程序首先需要一款工具。主要分为下面三大类

  • notepad(即windows里面的记事本)
    使用windows自带的记事本就可以写代码，只不过美观度、功能和效率上要远逊于下面的两种，不推荐。
  • Editplus、Notepad++、UltraEdit
    普通的文本编辑器，编写代码时具有关键字高亮功能，使代码更美观。初学者最好先使用这里面其中一款，有助于初学者学习。
  • Eclipse、MyEclipse、IDEA
    专业的Java开发集成环境（IDE），界面相比普通文本编辑器更加美观，智能化的代码提示，能够有效的提高工作效率，Eclipse免费，MyEclipse和IDEA收费。

建议大家先使用Editplus进行编辑，等学习深入之后再使用更高级的IDE进行开发。下载地址：http://pan.baidu.com/s/1qYVPfru

HelloWorld案例常见问题

  • 找不到文件
    • 文件扩展名隐藏导致编译失败，修改windows设置，显示所有文件名的后缀名，把文件名后缀.txt去掉。
    • 文件名写错了
  • 查看单词有没有拼写错误，Java区分大小写，注意大小写问题。
  • 检查是否缺少括号包括()和{}这些都应该是成对出现的。
  • 中英文问题，除了输入中文，其他所有文字和字符都必须是英文字符。
