集成开发环境

IDE是Integrated Development Environment的缩写，中文是集成开发环境的意思。在实际工作当中一般使用IDE进行开发，相比editplus的相关文本编辑器，IDE能够更进一步提高开发效率。目前企业里面应用比较多的IDE是下面三款：

  • eclipse（

    http://www.eclipse.org/）

    • 免费
    • 纯Java语言编写
    • 免安装
    • 扩展性强

  • MyEclipse（

    http://www.myeclipsecn.com/）

    • 在Eclipse基础上追加的功能性插件，对插件收费
    • 在WEB开发中提供强大的系统架构平台

  • IDEA（

    http://www.jetbrains.com/idea/）

    • 收费
    • UI比较美观

eclipse下载和安装

除了可以在eclipse官网下载，也可以在云盘里面下载：http://pan.baidu.com/s/1o7QZ8LW
注意分为32位和64位的。下载好之后， 将文件解压，之后双击eclipse.exe 。

使用eclipse编写HelloWorld

eclipse启动之后会提示选择工作空间，将来所编写的代码都存放在工作空间所在的目录里面。设置好工作空间之后，勾选上复选框，这样以后在启动eclipse时就不会再次弹出这个对话框了，点击ok

将欢迎界面关闭，进入到eclipse视图，点击file—>new—>project—>java project
)

输入项目名称（在eclipse里面是以项目为单位的），点击finish，在弹出的对话框里面点击yes。

在src上面右键，点击new—>package，创建一个包
)
在创建好的包上面右键，点击new—>class，填写类名，之后勾选上生成main方法的复选框

eclipse会自动编译源代码生成.class文件，不需要我们手动编译，所以编写好代码之后直接点击下图中的按钮即可运行（快捷键f11）。

eclipse会自动编译，如果编译不通过，会将错误显示出来，如下图：

如果出现黄色的提示，可以不用处理，程序依然能够正常编译和运行。这里的黄色提示只是一个警告不是错误：
