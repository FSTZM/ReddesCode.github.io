MySQL的安装与配置

阅读量:

MySQL的安装

MySQL下载：http://pan.baidu.com/s/1eSy6tk2
MySQL安装文档：https://pan.baidu.com/s/1IN7Bh5koYQc1WVks50D8uQ

提取码：yxnu
参考上面的安装文档一步步进行安装即可，安装过程较为简单，有两个地方需要注意，设置字符编码时，需要设置为UTF-8，在设置密码时，最好设置为好记的，我这里设置为：monkey1024

MySQL基本命令

启动和停止服务：
MySQL会开启一个服务，默认会启动这个服务，如果没有启动的话，可以手动开启：
启动和停止方式一：
右键我的电脑–>管理—>服务和应用程序—>服务—>找到名称是mysql开头的（有的是mysql55），右键启动或停止即可。

启动和停止方式二：
按下快捷键win+r，输入services.msc,找到名称是mysql开头的（有的是mysql55），右键启动或停止即可。

启动和停止方式三：
打开命令提示符（管理员权限的），
在里面输入:net start mysql55(如果失败的话，输入:net start mysql)启动。
输入：net stop mysql55(如果失败的话，输入：net start mysql)停止。

登录
打开命令提示符，输入下面命令：mysql -u root -p
按下回车键，会提示输入密码，输入MySQL的登录密码即可。
如果要登录远程的mysql，需要输入下面的命令：mysql -h 远程mysql服务器的ip地址 -u root -p
例如：mysql -h 192.168.0.1 -u root -p

注：如果在命令提示符中登录mysql时提示”mysql不是内部或外部命令时”，需要将mysql的安装目录（选择到bin目录，我本地的路径是：D:\Program Files\MySQL\MySQL Server 5.5\bin）加入到环境变量path中。

忘记密码
如果忘记root用户的登录密码可以使用下面的方法进行重置（仅供学习，工作中最好不要使用）：

  1. 停止mysql服务
  2. 打开命令提示符（管理员权限的），输入：mysqld –skip-grant-tables 按下回车键（不要关闭该窗口）
  3. 新打开一个命令提示符（管理员权限的），输入：mysql -u root -p 提示输入密码时，什么都不输入，直接按下回车键，进入mysql中。
  4. 输入：use mysql;
  5. 输入：update user set password=password(‘monkey’) WHERE User=’root’;其中的monkey是要设置的新密码。
  6. 关闭两个命令提示符，在任务管理器中将mysqld.exe的进程杀掉，之后启动mysql服务登录即可。
