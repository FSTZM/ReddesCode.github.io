子查询

定义：select 语句嵌套 select 语句被称为子查询；
注意：select子句可出现在select、from、where关键字后面，如下：
select … (select)…【使用较少，了解即可】
from …(select)…
where …(select)…

可以将select语句查询出的数据看做是一张新的表。

limit

作用：获取一表前几条戒中间某几行数据，主要用来分页处理，limit关键字只在MySQL中起作用。

用法：limit 起始下标m,长度n，
m：记录开始的index，默认从 0 开始，表示第一条记录；
n ：指从第 m+1 条开始，取 n 条；
