如果想要约束xml中的标签内容，可以使用下面两种方式编写约束内容：

dtd

dtd的全称是document Type Definition，即文档类型定义。
dtd的分类：

  • 内部dtd：在xml内部编写dtd
  • 外部dtd：在外部文件中编写dtd，然后再引入到xml中
